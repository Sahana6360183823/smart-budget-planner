# Smart Budget Planner 💰

A MERN Stack based web application that helps users manage their monthly income and expenses efficiently.

## Features

- Add monthly income
- Add expenses with name and amount
- View total expenses
- Calculate remaining balance
- Delete expenses
- Display expense distribution using pie chart

## Technologies Used

### Frontend
- React.js
- Vite
- CSS
- Recharts (Pie Chart)

### Backend
- Node.js
- Express.js

### Database
- MongoDB
